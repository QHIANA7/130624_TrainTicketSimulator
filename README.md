### 130624_TrainTicketSimulator
기차 발권 시뮬레이트 프로그램

### 개요
* 출발역과 도착역 선택 후 특정 열차 선택시, 여정 시간과 운임 출력이 가능합니다.
  
### 개발 목적
* 학과에서 배운 C를 이용해서 처음으로 만든 프로젝트로 기존에 동경하던 기차와 연관시켜 응용해보고자 개발했습니다.

### 개발 기간
* 2013.06.24. ~ 2013.07.04

### 주요 기술
* 참여도: **개인프로젝트**
* 개발 언어 : C
* 개발 환경 : Windows 10, Visul Studio 2017
